## 🎯 Objetivo

## 📚 Estrutura do Curso
O curso está dividido em quatro módulos progressivos:
1.  **Módulo 1: Fundamentos de Python** (Sintaxe, Variáveis e Lógica)
2.  **Módulo 2: Manipulação de Dados** (Introdução ao Pandas e Limpeza)
3.  **Módulo 3: Visualização de Dados** (Gráficos com Matplotlib e Seaborn)
4.  **Módulo 4: Análise Exploratória (EDA)** (Projeto Prático Final)

## 🚀 Como utilizar este material
Este curso foi desenhado para ser executado no **Google Colab**, o que dispensa a instalação de softwares no seu computador.
1.  Acesse a pasta do módulo desejado.
2.  Clique no arquivo `.ipynb` (Notebook).
3.  Utilize o botão "Open in Colab" (se disponível) ou faça o download e carregue no [Google Colab](https://colab.research.google.com/).
